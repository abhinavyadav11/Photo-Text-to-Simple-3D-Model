# Photo/Text → 3D Model Prototype

## Overview
This prototype demonstrates two pipelines:

- **Photo**: single-image segmentation + 2.5D extrusion → `.obj`/`.stl`
- **Text**: stub text-to-box placeholder → `.obj`/`.stl`
